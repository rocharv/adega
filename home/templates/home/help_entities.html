Entidades

Nesse seção vamos esclarecer o funcionamento de cada uma das entidades do sistema Adega.

Clique em uma delas para ver mais detalhes:

Um endereço é a descrição de um local que será associado ao cadastro de uma pessoa ou empresa.

Dica: Se você estiver cadastrando um endereço no Brasil, comece pelo CEP. O sistema Adega vai procurar os dados disponíveis, a partir do CEP, e preencherá automaticamente os campos restantes (geralmente com exceção do número e complemento, todos os outros campos).

Pessoas Físicas (Pessoas)

Uma pessoa é qualquer indivíduo que precise estar cadastrada no sistema. Ela pode ser uma funcionária, uma colaboradora, uma fornecedora, uma cliente, etc.

Pessoas Jurídicas (Empresas)

Uma empresa é qualquer entidade que precise estar cadastrada no sistema. Ela pode ser uma empresa, uma instituição, um órgão público, etc.

Um item é qualquer coisa que possa ser gerenciada em um armazém. Ele pode ser um produto, um equipamento, um material, uma obra de arte, uma licença digital de software, etc. Porém, todo item deve ter uma maneira de ser unicamente identificado, seja por um número de série, uma etiqueta patrimonial ou, no limite, o id gerado pelo sistema.

Todo item deve ser criado a partir de uma categoria de itens. A categoria de itens é uma abstração que define as características comuns de um grupo de itens.

Por exemplo, a categoria de itens com o nome "Apple iPhone 16 Pro Max - Titânio Preto" define perfeitamente um tipo de celular, mas não define um item armazenável. Um item precisa ter sido criado a partir de uma categoria de itens e conter um número exclusivo (dado por uma etiqueta patrimonial, número de série ou id). Pense que a empresa pode ter comprado 3 iPhones iguas, mas cada um deles é um item diferente. O mesmo vale para computadores, impressoras, etc. Telefones celulares, por exemplo, possuem IMEI diferentes, que pode ser cadastrado como um número de série.

Uma categoria de itens não pode ser incluída ou removida de um armazém. Mas existe um tipo especial de categoria de itens, para os itens chamados fungíveis. Itens fungíveis são aqueles que não precisam ser individualmente identificados, como papel, canetas, grampos, etc. Quando você criar uma categoria de item fungível, ela automaticamente cadastrará um item padrão com essa categoria.

Um armazém, na perspectiva do sistema, é qualquer local onde possam ser gerenciados itens.

Ele pode ser um almoxarifado, um depósito, uma sala, um armário, etc.

O armazém padrão nada mais é do que aquele que aparecerá por padrão, pré-selecionado, em todos os campos de seleção de armazém do sistema. Ele pode ser alterado a qualquer momento.

Dica: Não é recomendada a criação indiscriminada de armazéns, já que isso pode tornar mais lenta a gestão do inventário. O ideal é criar armazéns apenas para locais que realmente necessitem de controle.

As transações de entrada e saída são registradas no sistema para controlar o fluxo de itens nos armazéns.

  1. Escolha se quer incluir uma transação de entrada ou saída de estoque.
  2. Escolher o armazém envolvido.
  3. Selecionar o tipo de transação. As de entrada são: compra, doação, empréstimo, retorno e transferência. As de saída são: venda, doação, empréstimo, retorno, transferência e descarte.
  4. Adicione o número da nota fiscal ou recibo, caso exista.
  5. Selecione a quantidade.
  6. Selecione o item.
  7. Selecione o preço.
  8. Selecione a contraparte. Contrapartes podem ser pessoas e/ou empresas. Por exemplo: se a transação for uma compra, a contraparte será o fornecedor.
  9. Selecione a data de início, opcionalmente quando ocorreu o início transação.
  10. Selecione a data de término, opcionalmente quando ocorrerá o término transação. Por exemplo: digamos que você está fazendo uma operação de empréstimo de um livro para um colaborador. Você pode colocar aqui a data limite para a devolução.
  11. Selecione uma descrição da transação, se quiser.

Os Relatórios de Estoque são gerados a partir das transações registradas no sistema. Eles consolidam a posição do armazém, somando todas as transações de entradas ocorridas e subtraindo todas as saídas ocorridas.

Quando selecionado o Estoque por Categoria, a agregação é feita por categoria de itens. Ou seja, itens diferentes, mas que pertencem à mesma categoria de itens, são somados e subtraídos.

Já quando selecionado o Estoque por Itens, são mostrados os itens remanescentes no estoque, individualmente.
