Entidades

Nesse seção vamos esclarecer o funcionamento de cada uma das entidades do sistema Adega.

A Adega ajuda a gerenciar:

  • Endereços
  • Pessoas Jurídicas e Físicas
  • Categorias de Itens & Itens Individuais
  • Transações de Entrada e Saída
  • Relatórios de Estoque

Armazéns

Um armazém, na perspectiva do sistema, é qualquer local onde possam ser gerenciados itens.

Ele pode ser um almoxarifado, um depósito, uma sala, um armário, etc.

O armazém padrão nada mais é do que aquele que aparecerá por padrão, pré-selecionado, em todos os campos de seleção de armazém do sistema. Ele pode ser alterado a qualquer momento.

Dica: Não é recomendada a criação indiscriminada de armazéns, já que isso pode tornar mais lenta a gestão do inventário. O ideal é criar armazéns apenas para locais que realmente necessitem de controle.

Pessoas

Uma pessoa é qualquer indivíduo que precise estar cadastrada no sistema. Ela pode ser uma funcionária, uma colaboradora, uma fornecedora, uma cliente, etc.

Empresas

Uma empresa é qualquer entidade que precise estar cadastrada no sistema. Ela pode ser uma empresa, uma instituição, um órgão público, etc.

Itens & Categoria de Itens

Um item é qualquer coisa que possa ser gerenciada em um armazém. Ele pode ser um produto, um equipamento, um material, uma licença digital de software, etc. Porém, todo item deve ter uma maneira de ser unicamente identificado, seja por um número de série, uma etiqueta patrimonial ou, no limite, o id gerado pelo sistema.

Todo item deve ser criado a partir de uma categoria de itens. A categoria de itens é uma abstração que define as características comuns de um grupo de itens.

Por exemplo, a categoria de itens com o nome "Apple iPhone 16 Pro Max - Titânio Preto" define perfeitamente um tipo de celular, mas não define um item armazenável. Um item precisa ter sido criado a partir de uma categoria de itens e conter um número exclusivo.

Uma categoria de itens não pode ser incluída ou removida de um armazém.

Dica: A vantagem de criar uma categoria de itens é que ela permite o cadastro em massa de diversos item no armazém, cada um com seu id de sistema. Posteriormente, é possível entrar no armazém e editar detalhes dos itens individualmente sempre que necessário, atribuindo-lhes um número de série, etiqueta patrimonial etc.
